# piggy treeview

A Treeview component for Flutter, featuring expand/collapse all, search, hilite, checkbox.

## Getting Started

How to use? 
1. Import TreeView.dart.
2. Derive TreeNodeData to publish your own data.
3. See AccountList.dart for the example detail.
